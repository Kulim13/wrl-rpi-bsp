Raspberry Pi BSP
================

1. Supported Boards
-------------------

This BSP supports the following boards:

  * Raspberry Pi

Untested, but these should also work:

  * Raspberry Pi Zero
  * Raspberry Pi Zero Wireless
  * Compute Module
